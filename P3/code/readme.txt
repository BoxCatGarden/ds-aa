本工程使用VC++6.0搭建、编译 默认设置 Windows10环境下运行正常
1.进入VC++6.0，新建空的控制台程序。
2.选择顶部菜单栏“Project -> Add To Project -> Files”，将指定文件添加到工程。
  如果建立demo程序，添加：除"main_case.cpp"以外的所有cpp文件和h文件。
  如果建立测试程序，添加：除"main.cpp"以外的所有cpp文件和h文件，以及../documents/test files下所有文件。
3.编译，直接运行。

注意事项：Shortest_Path_case.exe需要和../documents/test files下所有文件放在一起运行。
